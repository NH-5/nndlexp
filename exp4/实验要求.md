## 模型二：Vision Transformer

请实现 ViT 模型，文件位置：
```python
models/vit.py
```
优先方案：

使用 torchvision.models.vit_b_16。

要求：

* 加载 ImageNet 预训练权重；
* 替换最后的分类头，使输出类别数为 10；
* 模型类名或函数名为 build_vit_model。

示例思路：
```python
from torchvision.models import vit_b_16, ViT_B_16_Weights
import torch.nn as nn
def build_vit_model(num_classes=10, pretrained=True):
    if pretrained:
        weights = ViT_B_16_Weights.DEFAULT
    else:
        weights = None
    model = vit_b_16(weights=weights)
    in_features = model.heads.head.in_features
    model.heads.head = nn.Linear(in_features, num_classes)
    return model
```

## 实验分析要求

请在 report.md 中加入以下分析点：

1. CNN 更擅长提取局部特征；
2. ViT 将图像划分为 patch，并把 patch 当作 token；
3. ViT 通过 self-attention 建模图像不同区域之间的关系；
4. 在小数据集上，直接训练 ViT 可能不如 CNN 稳定；
5. 使用预训练 ViT 可以提升收敛速度和分类效果；
